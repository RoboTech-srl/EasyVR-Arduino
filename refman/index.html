EasyVR Library Documentation

The EasyVR library implements the serial communication protocol to manage the EasyVR module and the EasyVR Shield from Arduino boards and controllers and it enables easy access to all the EasyVR features.

EasyVR module
EasyVR Shield

Installation

To install the EasyVR library on your Arduino IDE use the menu Sketch > Import Library ... > Add Library and open the released zip archive.

Examples

You can easily open the example sketches included with the EasyVR library from inside the Arduino IDE, using the menu File > Examples > EasyVR and choosing one of the available sketches.
